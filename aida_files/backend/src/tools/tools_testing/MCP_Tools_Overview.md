# MCP Tools Overview

| Tool Name                   | Category        | Description                                                                                   |
|----------------------------|----------------|-----------------------------------------------------------------------------------------------|
| **CurrentDateTool**        | Product Owner   | Provides the current date and time for scheduling, planning, and time-based operations.        |
| **ProjectBoardDataFetcherTool** | Product Owner   | Retrieves all tasks, cards, stories, and activities from a project management board.           |
| **TaskDataFetcherTool**    | Product Owner   | Fetches detailed information about a specific task/card, including status and history.         |
| **BoardCardManagerTool**   | Product Owner   | Manages cards on project boards: create, list columns, and provide usage guidance.             |
| **JiraProjectManagerTool** | Product Owner   | Manages JIRA projects and issues: list projects, create issues, and fetch project details.     |
| **FileReadTool**           | Developer       | Reads the content of a file and returns it as a string for downstream processing.              |
| **QueryOptimizerTool**     | Developer       | Analyzes and optimizes SQL queries for performance and readability across multiple dialects.   |
| **ConvertSQLQuery**        | Developer       | Converts SQL queries from one dialect to another using SQLGlot.                                |
| **extract_sql_information**| Developer       | Extracts structural information from a SQL query: tables, columns, joins, filters, etc.        |
| **GenerateSQLDiagram**     | Developer    | Generates a diagram connecting source tables to a target table using Matplotlib.               |
| **WebSearchTool**     | Architecture    | Check internet for Architeture code standards.               |
